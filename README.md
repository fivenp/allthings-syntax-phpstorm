# Allthings Syntax Theme

### PhpStorm
**OSX**
```sh
$ cp Allthings.icls ~/Library/Preferences/<Product>/colors/
```
**Windows**

Copy `Allthings.icls` to `C:/Users/USERNAME/.<Product>/config/colors`

**Linux**
```sh
$ cp Allthings.icls ~/.<Product>/config/colors/
```
